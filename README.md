# HipSlack

This project is generated with [yo angular generator](https://github.com/yeoman/generator-angular)
version 0.12.1.

## Install

set config.json

```config.json
{
	key: '{API_KEY}'
}
```

### Windows
%APPDATA%/hipslack/config.json

### OS X
~/Library/Application Support/hipslack/config.json


## Build 

- Run `gulp package`

## development

- Run `gulp serve` for preview.
